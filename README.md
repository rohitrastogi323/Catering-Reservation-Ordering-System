# Basic Calculator

## Project Description
This project is a basic calculator created using HTML, CSS, and JavaScript. 
It performs arithmetic operations such as addition, subtraction, multiplication, and division.

---

## Features
- Responsive design (mobile, tablet, desktop)
- Addition, Subtraction, Multiplication, Division
- Decimal support
- BODMAS-compliant calculations
- Keyboard input support
- Error handling (division by zero, invalid input)

---

## Technologies Used
- HTML5
- CSS3
- JavaScript

---

## How to Use
1. Open `index.html` in any browser
2. Click buttons or use keyboard
3. Press Enter to calculate
4. Press C to clear

---

## Keyboard Shortcuts
- Numbers: 0–9
- Operators: + - * /
- Enter: Calculate
- Backspace: Delete
- C: Clear
